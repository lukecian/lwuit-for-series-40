Hands-on Exercise

  1. Begin by downloading the design document, lwuit_hol.pdf. Read the document to understand the application you are trying to build.
  2. Next, get the skeleton project, ThemeStore.zip. Unzip the file and open the project in NetBeans. Adjust the path to LWUIT and make sure your emulator is set up correctly. Once everything is in place, you will be able to build and run the project, although it won't do much.


A complete, running solution to the ThemeStore exercise is available, ThemeStore-solution.zip.
